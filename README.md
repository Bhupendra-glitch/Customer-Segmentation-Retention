# 📞 Telco Customer Churn Prediction

## 📌 Project Overview
Built a machine learning classification model to predict customer churn using telecom customer data.

## 📊 Dataset
- 7,000+ customer records
- 20+ features including tenure, contract type, monthly charges

## 🎯 Objectives
- Identify key churn drivers
- Handle class imbalance
- Improve prediction performance

## 🔍 Model Implementation
- Logistic Regression
- Random Forest
- XGBoost
- LightGBM
- Stacking Model
- SMOTE for class imbalance handling

## 📊 Performance
- Achieved 80%+ accuracy
- F1-score (Churn class): 0.60–0.70
- Stacking improved performance by ~0.01

## 🛠️ Tech Stack
- Python
- Pandas
- Scikit-learn
- XGBoost
- LightGBM
- Matplotlib / Seaborn

## 🚀 Business Impact
- Identified high-risk churn customers
- Supported data-driven retention strategies
